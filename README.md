# Space-invaders
This space indvaders game is made in python using the pygame module. With the help of a youtube video https://www.youtube.com/watch?v=FfWpgLFMI7w&t=3987s .
There are a few changes that are made. To make the game more enjoyable. Well free to take the code and modify it.

# System requirements
* python
* pygame

# Installation
In order to play the game you must have python, preferably the latest version, and must have pygame also installed.
Once you check that your pygame is working correctly just execute the main.py file and you would be good to go.
